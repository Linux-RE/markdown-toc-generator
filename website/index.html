Effortless Table of Contents for your Markdown files

🚀 Get Started

🌟 Overview

@rrtechpro/markdown-toc-generator is a lightweight, flexible tool to generate stylish and accurate Table of Contents (TOC) from Markdown headers. Whether you want HTML or Markdown output, it’s built to fit seamlessly into your documentation workflow.

⚙️ Features

  • Auto-generate TOC based on headings
  • HTML and Markdown output formats
  • "Back to Top" link support
  • Customizable numbering styles (HTML only)
  • Anchor slug generation for internal linking
  • Fully customizable title and footer

📦 Installation

npm install @rrtechpro/markdown-toc-generator

🛠️ Usage Example

import { generateTOC } from '@rrtechpro/markdown-toc-generator'; const markdownContent = ` # Introduction ## Getting Started ### Requirements ## Installation # Usage `; const options = { format: 'html', tocTitle: { level: 2, title: 'Table of Contents' }, addBackToTop: true, tocFooter: 'Generated by My Tool', anchorPrefix: 'docs/' }; console.log(generateTOC(markdownContent, options));

🔧 Options

Option Type Description
format "html" | "markdown" Output format for the TOC
tocTitle { level: number, title: string } Heading level and title text for TOC
addBackToTop boolean Adds "Back to Top" link at end
numberingStyle "numeric" | "roman" | "alphabetical" (HTML only) Numbering style
tocFooter string Footer note for the TOC
